# Dotfiles Readme

My dotfiles. Primaraly set up to manage my vim config.

## Setup

    cd ~
    git clone https://github.com/olorton/dotfiles.git ~/.dotfiles
    ln -s ~/.dotfiles/bashrc ~/.bashrc
    ln -s ~/.dotfiles/bash_profile ~/.bash_profile
